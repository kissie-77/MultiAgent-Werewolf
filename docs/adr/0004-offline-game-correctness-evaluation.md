# ADR-0004: 离线游戏正确性评测独立成模块

**日期**: 2026-05-18 · **状态**: accepted

## 问题
评测要同时读取引擎事件、状态快照、观察视角和报告产物。若散落在 CLI 或核心引擎中，会让游戏逻辑和实验逻辑耦合。

## 决定
新增 `llm_werewolf.evaluation` 独立模块，复用 `GameEngine`、事件日志和状态序列化；通过 `werewolf-eval` 离线运行场景并输出报告。

## 取舍
- 付：第一版 checker 依赖事件结构，部分语义只能标记为结构缺失。
- 弃：直接做 Web 复盘或模型排行榜；先保证系统正确性评测可重复。
