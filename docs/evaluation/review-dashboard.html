PostGame 流水线（14 步）

分步 try/except 执行；单步失败不中断后续可独立步骤；仅 load_context 为必需步骤。 camp_persuasion 失败会跳过 log_views → coach，但 episodic、vote_swing、game_quality_report 仍执行。

质量门控（2026-06-02）

  • roster：从 message / hunter_revenge 等事件补全 role；缺 camp 回填
  • matched：阵营有利出局才算；误出同伴不算；无 swing 但意向=出局者也算
  • MVP：缺 role/camp 不参与归一化；全场 MVP 优先有身份玩家
  • proposals / skills：正向素材需 matched；误出 Seer 等 → bad_case
  • evolver：golden / positive 提案 evidence 须含 matched 才采纳

详见 docs/evaluation/DESIGN.md §5.2–§5.3

📢 Persuasion (公开说服)

仅白天公开发言、放逐投票与公开频道投票意向；不含狼队夜间讨论与私密技能。

包含事件: player_speech, vote_cast, vote_result, vote_intention_snapshot (public), player_eliminated, phase_change 过滤规则: - 排除 visible_to 非空的事件 - 排除 player_discussion - 仅白天阶段 (day_discussion, day_voting)

🐺 Wolf Night (狼队夜间)

仅狼队频道发言、狼队投票意向与当晚刀口结果；不含白天公开讨论。

包含事件: player_discussion (visible_to ⊆ wolves, ≥2人) werewolf_killed wolf_team 频道 vote_intention 结构化数据: kills_by_round: [{round_number, target_id, target_name}]

🎯 Strategy (角色策略执行)

仅技能判定、夜间刀口、放逐投票等行为事件；不含公开发言正文。

包含事件: seer_checked, witch_saved, witch_poisoned, guard_protected, werewolf_killed, vote_cast, role_acting 过滤规则: - player_discussion 仅包含狼队夜间讨论 - 排除公开发言正文

🏆 Outcome (结果与归因)

仅投票、出局、死亡、刀口与终局胜负；不含发言与意向细节。

包含事件: vote_cast, vote_result, player_eliminated, player_died, game_ended, werewolf_killed, phase_change 过滤规则: - 最精简的事件集 - 不含任何发言内容 - 不含投票意向

统计方法

Wilson 置信区间: 小样本时比简单 p ± 1.96*sqrt(p*(1-p)/n) 更准确 双比例 Z 检验: 判断 B 的胜率提升是否是统计显著的 pooled = (wins_a + wins_b) / (games_a + games_b) z_score = (phat_b - phat_a) / SE p_value = 2 * (1 - Φ(|z_score|)) 推荐逻辑: if win_delta > 0.05 and completion_b >= completion_a and significant: recommendation = "recommend_b" elif completion_b + 0.05 < completion_a: recommendation = "reject_b" else: recommendation = "no_clear_winner" 注意: win_rate = 有 winner_camp 的局占比，非阵营胜率
