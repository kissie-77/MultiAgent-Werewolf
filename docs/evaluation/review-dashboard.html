排行榜系统架构

📊
Entry 构建
从 run 目录提取指标：胜率、完成率、MVP评分、平均轮次等
🏆
Leaderboard 聚合
按胜率排序，生成总榜和分组榜（按 model/prompt/skill）
🔬
A/B 对比
Wilson 置信区间、双比例 Z 检验、p-value、推荐结论

统计方法

Wilson 置信区间: 小样本时比简单 p ± 1.96*sqrt(p*(1-p)/n) 更准确 双比例 Z 检验: 判断 B 的胜率提升是否是统计显著的 pooled = (wins_a + wins_b) / (games_a + games_b) z_score = (phat_b - phat_a) / SE p_value = 2 * (1 - Φ(|z_score|)) 推荐逻辑: if win_delta > 0.05 and completion_b >= completion_a and significant: recommendation = "recommend_b" elif completion_b + 0.05 < completion_a: recommendation = "reject_b" else: recommendation = "no_clear_winner"
