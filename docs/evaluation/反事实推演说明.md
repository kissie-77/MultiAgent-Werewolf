# 反事实推演说明

反事实推演用于回答：“如果当时做了另一个关键行动，局势可能会怎样？”

当前实现位置：

- `src/llm_werewolf/evaluation/post_game/counterfactual.py`
- PostGame pipeline 自动写出 `counterfactual_report.json`
- PostGame pipeline 自动写出 `counterfactual_report.md`

## 当前覆盖

- 近票差放逐：如果少量玩家改票，放逐目标是否可能改变。
- 狼人夜间刀口：如果夜间目标换人，白天讨论和生存节奏可能如何变化。
- 预言家查验：如果查验另一个高怀疑位置，公开信息价值可能如何变化。

## 边界

这一版不是完整分支模拟器，不会重新跑一整局游戏。它是稳定、可解释的规则型反事实报告，适合评分和答辩展示。

后续如果要冲更高，可以把每个反事实 case 接到轻量模拟器或 LLM 复盘，让系统生成更细的“替代时间线”。
