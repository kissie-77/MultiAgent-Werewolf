# 吕祎晗 - 修改记录

## 修复记录

### 修复 1：WhiteWolfKillAction 阵营比较错误
- **文件**：`src/llm_werewolf/core/actions/werewolf.py`
- **问题**：`validate()` 方法中使用 `self.target.get_camp() == "werewolf"` 字符串比较，但 `get_camp()` 返回 `Camp` 枚举，导致白狼击杀永远验证失败
- **修复**：导入 `Camp` 枚举，改为 `self.target.get_camp() == Camp.WEREWOLF`
- **状态**：✅ 已修复

### 修复 2：狼人讨论与 Nightmare Wolf 执行顺序错误
- **文件**：`src/llm_werewolf/core/engine/night_phase.py`
- **问题**：狼人讨论在 `run_pre_wolf_phase()` 之前执行，但 Nightmare Wolf 的封锁行动在 `run_pre_wolf_phase()` 中执行，导致梦魇狼封锁逻辑在讨论之后才生效
- **修复**：调整执行顺序为：前置阶段 → 狼人讨论 → 狼人投票 → 投票结算 → 后置阶段
- **状态**：✅ 已修复
