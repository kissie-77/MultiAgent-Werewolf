# 丘比特首夜连线目标选择

## 提取依据
[initial_curated] 丘比特首夜连线一对恋人，恋人同生共死、共同获胜；跨阵营连线更有利于第三方控场。

## 公开行为
① 优先连线一狼一好，制造阵营对立与不确定性；② 若身份难判，连线发言立场差异最大的两人；③ 连线后白天低调观察，勿暴露丘比特身份。

## 避免
① 不要连线两个明显同阵营玩家（除非有明确策略）；② 不要连线后立即暴露自己是丘比特；③ 不要忽视恋人存活对第三方胜利的重要性。
