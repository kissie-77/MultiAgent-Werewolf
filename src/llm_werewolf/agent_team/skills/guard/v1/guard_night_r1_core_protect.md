# 首夜守护核心位

## 提取依据
[initial_curated] 首夜无公开发言，守卫应预判狼队首刀规律守护高价值位，避免神职首夜裸倒。

## 公开行为
① 首夜优先守护高置位、或按板子规律最可能被刀的神职位（预言家/女巫）；② 遵守不能连续两晚守同一人的规则，首夜不必守自己；③ 第二天白天谨慎透露守护逻辑，仅在能形成排坑收益时说明。

## 避免
① 不要随机守护划水位；② 不要首夜守护全场都不信的跳身份位（可能是狼做身份）；③ 不要连续两晚守同一人；④ 避免无收益裸跳守卫。
